- `node down.js 2020-02-24` 下载指定日期数据
- `node pack.js 2020-02-24` 打包成 json 并清洗
- `node detail.js 2020-02-24` 获取成绩和榜单信息
- `node best.js 2020-02-24` 筛选最有价值成绩
